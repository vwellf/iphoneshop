Класи бутстрап для Iphone12

Navbar - Документація та приклади потужного, адаптивного навігаційного заголовка Bootstrap, панелі навігації. Включає підтримку брендингу, навігації тощо, включаючи підтримку нашого плагіна згортання.
navbar-brand - для назви вашої компанії, продукту чи проекту.
.navbar-nav - для повної та легкої навігації (включаючи підтримку спадних меню).
.navbar-toggler - для використання з нашим плагіном згортання та іншими способами перемикання навігації. Утиліти Flex та інтервалів для будь-яких елементів керування та дій форми.
.navbar-text - для додавання вертикально центрованих рядків тексту.
.collapse.navbar-collapse - для групування та приховування вмісту панелі навігації за батьківською точкою зупину.
.navbar-scroll -  щоб встановити максимальну висоту та прокручувати розгорнутий вміст панелі навігації.
.collapse приховує вміст
.collapsing застосовується під час переходів
.collapse.show показує вміст
Панелі навігації вимагають обгортання 
.navbar з .navbar-expand{-sm|-md|-lg|-xl|-xxl} для адаптивного згортання та класів колірної схеми.
